#### New Features

- **Max Trade Surplus BPS**: Added a new `maxSurplusBps` parameter for controlling max acceptable surplus on returned trades.
- **Dashboard SSO Login**: Added support for Single Sign-On (SSO) in the 0x Dashboard.

#### New Chains

- **Sonic**: Added Swap API support.
- **Unichain, Sonic, and Plasma**: Added Gasless API support.

- **Arbitrum:** Added Swaap.

- **Base:** Added Rubicon, 9mm.
- **BSC:** Added Swaap, Thena v2, Pancake Infinity.
- **Berachain:** Added Nabla.
- **Ethereum:** Added Rubicon.
- **Linea:** Added Swaap.
- **Mode:** Added Swaap.
- **Monad:** Added Magma, Kuru, Crystal, Nitro.
