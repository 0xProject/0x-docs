#### New Chains

- **Plasma Chain (9745)**: Now supported in Swap API.

#### New Features

- **Swap + Send**: Added support for swapping tokens and sending them to a specified address via a new `recipient` parameter in both Swap and Gasless APIs (see [API Reference](/api-reference/swap/).

#### New Liquidity Sources

- **Ethereum**: Added Fluid Lite.
- **Plasma**: Added Curve, Balancer v3, Fluid, and Uniswap v3.

#### Chores

- Uniswap v4 hook:
  - Added Renzo hook.
  - Updated Clanker, Flaunch, and Zora hooks.
- Improved routing for Zora ecosystem.
- Improved routing for Retake ecosystem.
- Improved latency across routing infrastructure.
