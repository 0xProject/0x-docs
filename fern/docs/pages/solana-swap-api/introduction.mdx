## What is the 0x Solana Swap API?

The 0x Solana Swap API is a fast, flexible API that lets developers programmatically access token prices across Solana's top DEXs, enabling seamless swap functionality in any app or wallet. It's designed to provide the same powerful aggregation and transaction-building features that 0x offers on EVM chains—now available on Solana.

Whether you're building a Solana-native experience or expanding an EVM-first product, the API helps you integrate token swaps with minimal overhead and maximum performance.

## Why Use 0x on Solana?

- **Unified Interface Across Chains** – Same quoting and swapping flow as 0x’s EVM API, making multi-chain support easy.

- **Built for Speed** – Leverages Solana’s fast block times for real-time swaps.

- **Competitive Pricing** – Aggregates liquidity from a growing set of Solana DEXs. While liquidity coverage is currently limited in alpha, we’re actively expanding coverage.

- **Developer-Friendly** – The API returns fully built, simulation-safe swap instructions—ready to sign and send.
