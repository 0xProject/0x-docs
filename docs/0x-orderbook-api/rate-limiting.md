# Rate Limiting

:::info


Rate limiting is used by the API to prevent abuse and ensure a reliable experience for all consumers. The current limit for the free tier of our APIs is approximately 2 Requests Per Second (RPS). Want higher rate limits? [Check out our enterprise options](https://0x.org/pricing)
 
To get live API key to access our APIs, visit the **[0x Dashboard](https://dashboard.0x.org)**. 

:::
