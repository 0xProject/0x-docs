# Getting Started

To create a account on the [0x Dashboard](https://dashboard.0x.org/apps), click **Sign Up**, enter your name, email address and password, and click **Continue**

## 3. Tour the Dashboard

### 3.1 Test API Key
Once you've created a team, you'll be taken to the [0x Dashboard](https://dashboard.0x.org/) where you can create and mange live API keys, find build resources, and raise support requests. 

To find a list of all supported networks by 0x, check out the [0x Cheat Sheet](https://0x.org/docs/introduction/0x-cheat-sheet).

Your dashboard comes with a Test API key you can use to make sample requests. 
Give it a try by plugging it into the curl request and running that from your CLI to see a live Swap API quote request: 


```bash
// Replace the API key in the header with your sample key
curl --location --request GET 'https://api.0x.org/swap/v1/quote?buyToken=DAI&sellToken=ETH&sellAmount=100000&excludedSources=Kyber' --header '0x-api-key: [api-key]'
```

Click **Create an app** to get a live API key that has access to multiple 0x products.

### 3.3 Add 0x Explorer Tag


In the next screen, you can add a unique tag that can be used to identify transaction data from your app on [0x Explorer](https://explorer.0x.org/apps). 

0x Explorer provides data about ERC-20 transactions routed through 0x API. It’s a reliable and transparent tool to help developers and users verify transactions and analyze on-chain activity in an easy and low-friction way.

[Learn more about 0x Explorer Tags](/docs/developer-resources/0x-explorer-tag-guide).

You can skip for now and add a tag later.

### 3.4 Reveal Your API Key

On the final screen, click the eye to reveal your API key! This key is unique and tied to your app. Do not share it. 

This key will allow you to authenticate requests on 0x. Remember to specify the key in your requests vai the `0x-api-key` header paramter:

For each Demo App, you can see the following (labels above correspond to list below):

1. See which 0x products are enabled - Swap API, Orderbook API.
2. From **API Key**, see the API key associated with this app and **Rotate the API Key**. From **Settings**, change the 0x products enabled for this app. Set or update your **0x Explorer Tag**. Change the **App Name**.
3. If you have set up an **0x Explorer Tag**, you can view your on-chain data and transactions related to this app. 
4. View **build resources**. 

## 5. Manage Your Account

You can find additional settings to manage your account from **Your Account** in the top-right corner. 
- **Settings** - You can see your full name, team name, and account email, as well as change your password. 
- **Contact Support** - Have a question? Reach out to our dev support team. 
- **Docs** - Jump into our [developer docs](https://0x.org/docs) and start building
- **Help** - Need help? Check out our commonly asked questions and help center.
- **0x Explorer** - If you have a [0x Explorer](https://explorer.0x.org/apps) tag setup, easily analyze your app's on-chain activity and transactions. [Learn more about the 0x Explorer Tag](/docs/developer-resources/0x-explorer-tag-guide).
