# Display swapped amounts

If you would like to easily display the the final amounts swapped after a 0x transaction has settled on the blockchain, we recommend using the [0x-parser library](https://github.com/0xProject/0x-parser/tree/main).

This library parses 0x transactions from EVM blockchains into a format that is user-friendly and easy to understand.
