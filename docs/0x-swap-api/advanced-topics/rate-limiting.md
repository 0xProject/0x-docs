# Rate Limiting

Rate limiting is used by the API to prevent abuse and ensure a reliable experience for all consumers. The current limit for the Free Tier of our APIs is approximately 1 Requests Per Second (RPS) and 200K API calls per month globally.

:::info
Want higher rate limits? [Check out our enterprise options](https://0x.org/pricing)
:::
